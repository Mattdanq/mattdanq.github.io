#### In order to run this website application for your own viewing on your own computer via a code editor/terminal:
1. install all files properly and make sure to enter the project folder using cd
2. execute ***npm install*** if you have not already (in terminal)
	
	There are separate items that you would need to npm install:
   * ***npm install*** react-router-dom
            
3. execute ***npm start*** to run the application in the project folder
4. This will open the website on your browser
